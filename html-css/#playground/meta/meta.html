Тег Описание Аттрибуты
meta Нужен для задания допольнительной информации о документе для браузера также этот элемент должен обязательно состоять из ключа и значения name or http-equiv | content | dir | lang | scheme
name
для указания имени свойства(ключ)
http-equiv
для указания имени свойства(ключ) вместо name
content
для указания значачения
dir
указывает направление текста
lang
указывает информацию о языке
scheme
указывает инструкции для обработки ключа и значачения

Code: 

        
          < meta
            < !--ключ  -->
            name="viewport" 
            < !-- значение -->
            content="width=device-width, initial-scale=1.0">
          < meta
             < !--ключ  -->
            http-equiv="X-UA-Compatible"
            < !-- значение -->            
            content="ie=edge">
            < !-- для отключения кеша -->
          < meta http-equiv="pragma" content="no-cache">
          < !-- для редиректа в указанное время на указанный адрес -->
          < meta http-equiv="refresh" content="0(time); url=http://example.com">
          < !-- для обновления страницы в указанное время -->
          < meta http-equiv="refresh" content="0(time);">
