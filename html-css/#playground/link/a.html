Тег Описание Аттрибуты
a Ссылка(гиппер ссылка) на другой документ или файл title | href | target | download

Code: 

    
        < a href="../../assets/img1.jpg" target="_blank">Link for image
        < a href="base.html">Link for html document
        < a href="../../assets/img1.jpg" download title="Image for download">Link for download image
        < a href="base.html" download="">Link for html document
