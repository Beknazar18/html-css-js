Тег Описание Аттрибуты
style Нужен для задания внутреннего стиля документа type="text/css" | media
type
указывает тип стиля
media
для указания целевого устройства (принтер, экран компьютера, мобильное устройство и т.д.)

Code: 

        
         < style type="text/css">
         body {
           background: yellow;
         }
