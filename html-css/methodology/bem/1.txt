БЭМ (блок элемент модификатор)
  - блок это самодастаточный (независимый)  компонент который может быть использован в любом месте html файла
  example: 
    -- <div class="error"></div>
    !неправильно!! <div class="red-text"></div>
    -- <div class="button"></div>
  Блок не должен влиять на свое окружение, т. е. блоку не следует задавать внешнюю геометрию (в виде отступов, границ, влияющих на размеры) и позиционирование.

  В CSS по БЭМ также не рекомендуется использовать селекторы по тегам или id.

  - элемент является частью блока 
  stx: block-name__element-name
  example: 
    --  <div class="error">
          <div class="error__logo"></div>
          <div class="error__text">
          </div>
          <div class="button"></div>
        </div>
  - модификатор определяет внешний вид элемента, поведение, состояние блока либо элемента
  stx:  block-name_modification
        block-name__element-name_modification

  Название модификатора характеризует внешний вид («какой размер?», «какая тема?» и т. п. — «размер»: size_s, «тема»: theme_islands), состояние («чем отличается от прочих?» — «отключен»: disabled, «фокусированный»: focused) и поведение («как ведет себя?», «как взаимодействует с пользователем?» — «направление»: directions_left-top).

  Имя модификатора отделяется от имени блока или элемента одним подчеркиванием (_)

  #Типы модификаторов
    Булевый (это модификатор который имеет значение true or false)
    stx:  block-name_modification
          block-name__element_modification
    Ключ-значение (это модификатор который может иметь другие значения кроме булевого)
    stx:  block-name_modification_value
          block-name__element_modification_value
